## Installation

```
git clone git://github.com/ricobl/dotfiles.git
./install.sh
```
### Operating system installation

#### Ubuntu

```
./ubuntu_install.h
```

#### MacOSX

```
./osx_install.h
```

To install YouCompleteMe you will need to run

```
cd ~/.vim/bundle/YouCompleteMe
./install.py --clang-completer --tern-completer
```
